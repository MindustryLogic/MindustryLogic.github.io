Structure or syntax

order based on appear order

＜!DOCTYPE html＞Must include,1 per page

The very first thing you'll see
It just tells the browser to render in "quirks mode"(whatever that means) and it just make it load faster

＜title＞Must include,1 per page

The title for ur page(thats it)

＜link rel="icon" href="" /＞Can be excluded,1 per page

This will attempt to find a image in href="" and us it as icon.If find nothing a error is show in debugger(F12) and browser use default

＜style＞ or ＜script＞Can be excluded,1 per page

This section is place for css if you don't want to use external css(and/or javascript)

Main Part

＜html＞Recommend to include,1 per page

This one CAN be excluded from source code.Usually paired with lang="" to specify language used

＜head＞Should not but can be excluded,1 per page

Contains metadata for the page,such as charset(character set used like utf-8)
Can also be used to jinx a css(like the source code of this page)

＜body＞Must include,1 per page

Where all the thing that will be show to people are included here.

＜h1,h2...h6＞Optional,unlimited

Heading 1 to 6.The higher the number the bigger the text is

＜p＞Optional,unlimited

The paragraph indicator.It exist so that you can style a lot of text with 1 single css rule.
